Syllabus

Month 1: Foundations and Fundamentals

Week 1-2: Introduction to Computers and Programming

  • What is a computer? Components and basic functioning.
  • Introduction to programming with C (flowcharts and pseudocode).
  • Basics of Python: Variables, loops, and conditionals.
  • Simple programs (e.g., a number guessing game).

Week 3-4: Exploring the Web and Logical Thinking

  • HTML & CSS: Basics and creating a simple webpage.
  • Logical reasoning (truth tables, basic logic gates).
  • Intro to Graph Theory: Nodes, edges, and real-world applications.
  • Hands-on: Drawing graphs for social networks or maps.

Month 2: Programming and Engineering Basics

Week 5-6: Advanced Python and Java

  • Functions, and file handling in Python.
  • Mini-project: A text-based game.
  • Introduction to Java: Syntax, variables, and loops.

Week 7-8: Computer Engineering Basics (Operating Systems)

  • How a computer processes data (binary numbers, CPU, memory).
  • Hands-on: Representing numbers and text in binary.
  • Intro to Operating Systems: What they do and why they matter.
  • Basic terminal commands (CLI tools).

Month 3: Networks, AI, and Web Development

Week 9-10: Networking and Web Development

  • Basics of networking (IP, protocols, how the internet works).
  • Building interactive web pages with JavaScript.
  • Project: Create a mini-website.

Week 11-12: AI and Cloud Computing

  • Introduction to AI: Applications and hands-on with online tools.
  • Introduction to Cloud Computing: Examples, how it works, and its benefits.

Month 4: Diving Deeper into Core Concepts

Week 13-14: Quantum Mechanics and Physics for Computing

  • Basic quantum mechanics concepts (qubits, superposition).
  • Fun with Schrödinger’s cat.
  • Applications of quantum mechanics in computing.

Week 15-16: Computer Organization and Architecture

  • How processors work (intro to architecture).
  • Hands-on: Understand a simple CPU model.
  • Memory and storage: RAM vs. ROM, and storage hierarchy.

Month 5: Final Projects and Integration

Week 17-18: Final Integration and Projects

  • Review of programming concepts (Python, Java, and C).
  • Project brainstorming: Building something of their choice.
  • Final Project 1: Web Development Showcase.

Week 19-20: Closing and Real-World Applications

  • Final Project 2: AI or Networking Simulation.
  • Integration: Discuss how concepts interconnect (networking, cloud, AI).
  • Presentations and feedback.
  • Career guidance and next steps to get into Tier 1 Universities.
